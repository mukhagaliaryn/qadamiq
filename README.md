# Qadam IQ
#### Балаларға арналған цифрлық оқу платформасы

## Қысқаша сипаттама
Бұл платформа бастауыш сынып оқушыларына арналған заманауи цифрлық оқу ортасы. Оқу материалы модульдер мен деңгейлерге бөлініп, әр оқушы өз қарқынымен жүйелі түрде білім алады.

Әр деңгейде оқушы:

- қысқа әрі түсінікті материалмен танысады
- интерактивті тапсырмалар орындайды
- ойын арқылы білімін бекітеді
- тест арқылы нәтижесін тексереді

Платформа оқушының прогресін автоматты түрде бақылап, келесі деңгейге өту үшін қажетті шарттарды ұсынады.

Қосымша AI тьютор оқушыға қиын сәттерде бағыт беріп, ойлануға көмектеседі, бірақ дайын жауапты бермейді.
